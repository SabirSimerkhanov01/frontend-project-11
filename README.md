### Install:
### Установка:
1. Clone repository with next comand:
```sh 
git clone https://github.com/SabirSimerkhanov01/frontend-project-11
```

2. Install proggram RSS aggregator use next commands:

```
make install
```

```
npm link
```

Compile webpack:

```
make build
```
